SEO Freelancer API

The API server is running correctly. This page serves as a placeholder for the frontend.

API Endpoints:

Bookings

GET /api/availability - Get available time slots for a date
GET /api/availability/next - Get the next available date
POST /api/bookings - Create a new booking
GET /api/bookings/:token - Get booking details by token
GET /api/bookings/confirm/:token - Confirm a booking

Admin

GET /api/admin/consultations - Get upcoming consultations
POST /api/admin/blocked-slots - Block a time slot

Blog

GET /api/blog-posts - Get published blog posts
GET /api/blog-posts/:slug - Get a specific blog post

For a complete list of endpoints, refer to the API documentation or check the server's README.
